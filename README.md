# ✊ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.  
The user competes against the computer, and the winner is decided based on classic Rock Paper Scissors rules.

## 📌 Features

- ✊ Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Computer generates random choices
- 🏆 Shows winner and game result
- 🎨 Clean and responsive user interface
- 🌙 Dark mode design
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎮 How to Play

1. Select Rock, Paper, or Scissors.
2. Computer will randomly choose its option.
3. Winner will be displayed instantly.
4. Enjoy the game!
